Book Structure#

Part I: Foundations#

Learn the basics of systematic trading, how markets work, and why human psychology makes manual trading so difficult.

Part II: Core Concepts#

Understand trend following, volatility, risk management, and the critical role of transaction costs.

Part III: Implementation#

Get hands-on with data acquisition, strategy building, and backtesting real trading systems.

Part IV: Analysis#

Master the art of interpreting results, calculating performance metrics, and building diversified portfolios.

Part V: Practical Application#

Learn what this system can and cannot do, how to extend it, and what to consider for real-world trading.

How to Use This Book#

Reading Online#

Simply navigate through the chapters using the sidebar. All code examples are displayed with their outputs.

Running Interactively#

Each chapter is a Jupyter notebook that you can run yourself:

  1. Open in Colab - Click the rocket icon at the top of any chapter

  2. Download and run locally - Clone the repository and run Jupyter

  3. Experiment freely - Modify examples and see what happens!

Prerequisites#

To just read: None! Start with Chapter 1.

To run the code:

  • Python 3.8+

  • Basic command-line familiarity (helpful but not required)

All Python concepts are explained as we go.
